# PetitesBouchees

## Project Overview

PetitesBouchees is a baby food diversification tracker (suivi de la diversification alimentaire) iOS app. Track which foods have been introduced, log reactions, and monitor progress across food categories. All UI text is in French only.

- **Bundle ID:** com.bnjdpn.PetitesBouchees
- **Team ID:** 767SX34A7Z
- **Deployment Target:** iOS 17.0+
- **Swift:** 6.0 (strict concurrency: complete)
- **iPhone only** (no iPad support)

## Technology Stack

Swift 6 ¬∑ SwiftUI ¬∑ SwiftData ¬∑ XcodeGen ¬∑ Fastlane ¬∑ GitHub Actions (self-hosted runner)

## Modern Swift/SwiftUI Conventions (MANDATORY)

All code MUST follow these conventions. Violations will be caught in review.

| Use | Never Use |
|-----|-----------|
| `@Observable` | `ObservableObject`, `@Published`, `@StateObject`, `@ObservedObject` |
| `NavigationStack` | `NavigationView` |
| `.foregroundStyle()` | `.foregroundColor()` |
| `.clipShape(.rect(cornerRadius:))` | `.cornerRadius()` |
| `Button` with label | `.onTapGesture` for interactive elements |
| `.onChange(of:) {}` (0 or 2 params) | `.onChange(of:)` with 1 parameter |
| `.containerRelativeFrame()` | `GeometryReader` (when possible) |
| `.task {}` | `onAppear` + `Task {}` |
| `if let` / `guard let` | Force unwrap `!` |
| `@MainActor`, `Sendable`, `sending` | Unstructured concurrency |
| `#Preview {}` macro | `PreviewProvider` |
| `struct` (default) | `class` (only for shared mutable state) |
| Hardcoded French strings | `String(localized:)` or `.xcstrings` (FR only, no localization needed) |

### Data Models

- **Food** -> name, category (FoodCategory), recommendedAgeMonths, notes
  - Relationship: `entries: [FoodEntry]` (cascade)
- **FoodEntry** -> dateIntroduced, reaction (Reaction), hasDigestiveIssue, hasAllergicReaction, notes
  - Relationship: `food: Food?`

### Enums

- **FoodCategory** (7 values): `legumes`, `fruits`, `feculents`, `proteines`, `produitsLaitiers`, `matieresGrasses`, `epicesAromates` ‚Äî each with French `displayName` and emoji
- **Reaction** (3 values): `bien`, `moyen`, `mauvais`

### Key Services

- **SeedDataService**: 191+ predefined foods across all 7 categories, inserted on first launch.
- **ScreenshotDataService**: Provides realistic seed data for Fastlane snapshot screenshots.

## Key Conventions

- **Data storage**: Age in months as Int. All user-facing text hardcoded in French (no localization framework needed).
- **Enums**: Store as raw `String` for future-proofing.
- **XcodeGen**: `project.yml` is the source of truth. Never edit `.xcodeproj` directly.
- **CI**: GitHub Actions triggers on `main` branch. Self-hosted macOS ARM64 runner, dynamic simulator selection, artifact upload on failure.
- **Testing**: Use Swift Testing (`@Test`, `@Suite`, `#expect`), not XCTest.
- **DEBUG mode**: Use in-memory ModelContainer for previews and tests.
- **Error handling**: Never use `try!` for ModelContainer ‚Äî always `do-catch`.
- **No localization**: App is French only. No `.xcstrings` file needed. Hardcode French strings directly.
- **No CloudKit**: Pure local SwiftData storage, no sync.

## Pre-Deploy Checklist

1. Version/build numbers updated in `project.yml`
2. No `#if DEBUG` guards on essential code (ModelContainer init, data migration)
3. No `try!` or `fatalError` in init/startup paths
4. Match provisioning profiles valid for bundle ID
5. Local release build tested on physical device
